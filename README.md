
# iNotebook

iNotebook is a note-taking web application that allows users to create, update, delete, and manage their notes securely.

## Features
 User Authentication (Login & Signup)
add Notes
Edit Notes
 Delete Notes
 Secure note management
 Responsive user interface

## Technologies Used
 Frontend
 React.js
Bootstrap
JavaScript

### Backend
 Node.js
  Express.js
MongoDB
